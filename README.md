# Prosocial

A resource platform for nonprofits, cooperatives, and socially-focused organizations
in the St. Louis region (Missouri & Illinois).

## Stack
- Next.js 14 (App Router)
- Neon (Postgres + pgvector)
- Drizzle ORM
- Anthropic Claude API
- Vercel Blob
- NextAuth.js
- Vercel (hosting)

## Getting Started

1. Clone the repo
2. Copy .env.local.example to .env.local and fill in values
3. Run `npx drizzle-kit push` to sync schema to Neon
4. Run `npm run dev` to start local dev server
